Max's Python project

My Python assignment was a code where the user puts in their height, and it will make a joke. After this it asks if the user is fine with another joke, and if they say yes, the code makes another one but if they say no, the code apologises and then it ends. The Python program is great for these types of tasks, it excels at IF and ELSE functions and it is a nice, little quick tool to help make small to big projects. If I had more time for the project, I could add in many more variables to make this game way more entertaining.
